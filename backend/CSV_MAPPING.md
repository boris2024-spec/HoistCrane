## Маппинг в модель Equipment

```python
CSV столбец → Equipment поле

"" (первый столбец) → equipment_number
"חברה" (название) → model
"סטטוס בדיקה" → определение status (active/inactive)
"סטטוס פрט ציוד" → определение equipment_type по ключевым словам
"???? ?????? ?????" → serial_number
"תאור" → description
"מיקום" → location_details
"אתר / סרק" → site_name
"מזמין" → employer
"יצרן" → manufacturer
"בודק/ת" → inspector_name  
"תקף עד" + последнее поле → парсинг дат last_inspection_date и next_inspection_date
```

## Типы оборудования (по ключевым словам в названии):

- "גני" / "Genie" / "lift" / "ליפט" → platform
- "מנוף" / "crane" → crane
- "מלגזה" / "forklift" → forklift
- "מעלית" / "elevator" → elevator
- Остальное → other

## Статусы:

- "תקף" / "בתוקף" / "פעיל" → active
- "לא תקף" / "לא בתוקף" → inactive
- По умолчанию → active
