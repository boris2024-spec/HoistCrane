# 🎨 Обновления дизайна Hoist & Crane

## ✨ Что нового

### 1. 📱 Responsive Header с Hamburger меню
- Современный header с полупрозрачным фоном (backdrop blur)
- Hamburger меню для мобильных устройств (до 960px)
- Плавные анимации открытия/закрытия меню
- Кнопка закрытия внутри мобильного меню

### 2. 🌓 Dark Mode
- Переключатель темной/светлой темы в header
- Сохранение выбранной темы в localStorage
- По умолчанию включен темный режим
- Плавные переходы между темами

### 3. 🎨 Современная цветовая схема
**Темная тема:**
- Фон: `#0a0f1a` → `#1a1f2e`
- Основной цвет: `#4ade80` (зеленый)
- Акценты: `#a3e635` (светло-зеленый)

**Светлая тема:**
- Фон: `#f8fafc` (очень светло-серый)
- Основной цвет: `#22c55e` (зеленый)
- Акценты: `#84cc16` (лайм)

### 4. 🦶 Профессиональный Footer
- 4 колонки с информацией:
  - О компании + социальные сети
  - Быстрые ссылки
  - Ресурсы
  - Контактная информация
- Copyright и дополнительные ссылки внизу
- Responsive дизайн для всех устройств

### 5. 💅 Улучшенный дизайн компонентов
- Скругленные углы (12px border-radius)
- Gradient для кнопок и заголовков
- Улучшенные тени и hover эффекты
- Современная типографика
- Кастомный scrollbar с зеленым акцентом

### 6. 🔐 Обновленная страница входа
- Градиентный фон с анимированными кругами
- Иконки в полях ввода
- Показать/скрыть пароль
- Улучшенная валидация

## 🚀 Как использовать

### Переключение темы
Нажмите на иконку солнца/луны в правом верхнем углу header.

### Мобильное меню
На мобильных устройствах (< 960px):
1. Нажмите на иконку ☰ (hamburger) в левом верхнем углу
2. Откроется боковое меню
3. Выберите нужный раздел или нажмите ✕ для закрытия

### Уведомления
Иконка колокольчика в header показывает количество непрочитанных уведомлений.

## 🎯 Технические детали

### Новые файлы
- `src/context/ThemeContext.js` - Управление темами
- `src/components/Layout/Footer.js` - Компонент footer

### Обновленные файлы
- `src/App.js` - Интеграция ThemeProvider
- `src/components/Layout/Layout.js` - Новый header и структура
- `src/pages/Login.js` - Современный дизайн
- `src/index.css` - Глобальные стили

### Breakpoints
- Mobile: `< 600px`
- Tablet: `600px - 960px`
- Desktop: `> 960px`

### Цветовая палитра
```css
/* Dark Mode */
--primary: #4ade80
--primary-light: #86efac
--primary-dark: #16a34a
--background: #0a0f1a
--paper: #1a1f2e

/* Light Mode */
--primary: #22c55e
--primary-light: #86efac
--primary-dark: #16a34a
--background: #f8fafc
--paper: #ffffff
```

Все используемые компоненты входят в стандартный Material-UI, дополнительные установки не требуются.
